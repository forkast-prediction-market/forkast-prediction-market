## Subscribe

Use the `events` channel to receive fills, claims, and dispute notifications.

## Event payloads

Each payload includes an event type, timestamp, and structured body tailored to the event.

## Idempotency

Use the delivered sequence number to detect duplicates when persisting events to your systems.
