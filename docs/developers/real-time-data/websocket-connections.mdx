## URL

Connect to `wss://api.forkast.xyz/v1/stream` with a standard WebSocket client.

## Authentication

Include your API key as a bearer token in the `Authorization` header when opening the socket.

## Heartbeats

Send a heartbeat every 30 seconds or listen for server pings to keep the connection alive.
