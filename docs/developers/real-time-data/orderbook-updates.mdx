## Subscribe

Send a subscription payload with `type: orderbook` and the market ID to receive incremental depth updates.

## Message format

Each message contains bids and asks represented as price-size tuples.

## Snapshots

Reconnects should start with a snapshot request to rebuild the book before applying deltas.
