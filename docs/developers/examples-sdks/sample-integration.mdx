## JavaScript quick start

Use the official SDK to initialize a client and list markets in under ten lines of code.

<Tabs items={['JavaScript', 'Python']}>
  <Tab>

```ts
import { createClient } from '@forkast/sdk'

const client = createClient({
  apiKey: process.env.FORKAST_API_KEY!,
})

const markets = await client.markets.list({ limit: 5 })
console.log(markets.items.map(market => market.slug))
```

  </Tab>
  <Tab>

```py
from forkast import Client
import os

client = Client(api_key=os.environ["FORKAST_API_KEY"])

markets = client.markets.list(limit=5)
for market in markets.items:
    print(market.slug)
```

  </Tab>
</Tabs>

<Callout type="success" title="Environment configuration">
  Store `FORKAST_API_KEY` in your hosting platform's secret manager. The same key
  powers automations that interact with {process.env.NEXT_PUBLIC_SITE_NAME}.
</Callout>
