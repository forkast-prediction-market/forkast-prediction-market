## Key scopes

Assign scopes per key to limit access to read-only, trading, or administrative actions.

## Rate limits

Each account receives a default rate limit measured in requests per minute. Exceeding the limit returns HTTP 429 responses.

## Requesting higher quotas

Contact support with your projected usage and business case to increase throughput.
