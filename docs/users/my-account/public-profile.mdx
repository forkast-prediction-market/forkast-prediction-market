## Overview

Your public profile showcases high-level stats:

- **Positions Value**, **Profit/Loss**, **Volume Traded**, **Markets Traded**
- Address short hash (e.g., `0xa4…9e542d`) and **join date**

---

## Tabs

- **Positions** – open holdings (if you opt to display them)
- **Activity** – recent buys/sells with amounts and timestamps

<Callout type="info">
You can access this page from **Profile Settings → View public profile**.
</Callout>
