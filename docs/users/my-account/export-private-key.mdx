## Export instructions

<Steps>
  <Step>
    ### Secure your environment

    Use a trusted device on a private network before exporting sensitive
    wallet data.
  </Step>
  <Step>
    ### Verify your identity

    Confirm your password and 2FA (if enabled) to authorize the export.
  </Step>
  <Step>
    ### Save securely

    Store the private key in an encrypted vault or password manager. Never
    share it via email, chat, or screenshots.
  </Step>
</Steps>

<Callout type="info">
Export is intended for wallet backup or migration. If you only need to change login methods, you usually don't need to export your key.
</Callout>
